## About this app

CleanShot X replaces the built-in screenshot tool with something far more powerful.

- **Capture Anything** — fullscreen, area, window, scrolling capture, and timed screenshots.
- **Screen Recording** — record screen as video or GIF with system audio and microphone support.
- **Quick Overlay** — captured screenshot floats on screen for quick annotation, copying, or dragging into other apps.
- **Annotation** — arrows, text, blur, highlight, numbering, and more.
