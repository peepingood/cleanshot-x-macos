Installation Guide

Step-by-step installation via Terminal

  1. ⌘

    Open Terminal

    Press ⌘ + Space to open Spotlight, type Terminal and press Enter.

  2. 2

    Install CleanShot X

    Paste this command and press Enter. Homebrew will download and install the app; you may be prompted for your Mac password.

    Terminal — Install CleanShot X
    $brew install --cask cleanshot
  3. ✓

    Done

    When the command finishes, open CleanShot X from Applications or Spotlight (⌘ + Space).

About this app

CleanShot X is the ultimate screen capture tool for Mac. Screenshots, scrolling capture, screen recording, annotation, OCR, and cloud upload.

CleanShot X replaces the built-in screenshot tool with something far more powerful — capture, record, annotate, and share from one place.

Capture Anything

Fullscreen, area, window, scrolling capture, and timed screenshots.

Screen Recording

Record screen as video or GIF with system audio and microphone support.

Quick Overlay

Captured screenshot floats on screen for quick annotation, copying, or dragging into other apps.

Annotation

Arrows, text, blur, highlight, numbering, and more with a built-in editor.
